Enumerable in Ruby is a module of useful methods that makes a Ruby programmer's life happy. There are many methods in the Enumerable Module. Each of those methods traverses a collection and do some pretty sweet things. For this post, I'll focus on the map method within the Enumerable module.

enum.map { |obj| block } â†’ array, the map method returns a new Array with the result of running the block once for every element in the enumerable. For example: 


          irb(main):001:0> my_array = [1, 2, 3, 4]
          => [1, 2, 3, 4]
          irb(main):002:0> my_array.map { |el| el + 1 }
          => [2, 3, 4, 5]

The map method can also selectively modify elements in an enumerable by adding a condition in the block. For example: 


          irb(main):004:0> my_array.map { |el| if el % 2 == 0 then el + 10 else el end }
          => [1, 12, 3, 14]

The enumerable module provides powerful and elegant methods to transform collections. Not only do those methods provide easy ways for the developer to manipulate collections, they also make the code very readable. Some of my other favorite methods from the enumerable methods are inject, partition, and group_by, be sure to check them out on the Ruby docs.
